# 解構：Block Blast!

## 核心迴圈（一圈幾秒？操作→回饋→獎勵怎麼轉？）

**一圈約 3–8 秒。**

```
看三個備選方塊（無時限）
       ↓
選一塊、拖曳到 8×8 格任意位置
       ↓
放下的瞬間：格子高亮確認位置
       ↓
整行或整列滿格 → 爆破消除 + 分數跳動 + 音效
       ↓
三塊用完自動補三塊新的
       ↓
格子不夠放任何一塊 → 遊戲結束
```

關鍵：**玩家不被催**（無落下計時），壓力來自「格子越來越滿」這個視覺威脅，而不是倒數計時——讓人感覺是在下棋，不是在搶拍。

---

## 上癮機制（心理學拆解）

| 機制 | 怎麼觸發 |
|---|---|
| **Near-miss（差一點）** | 一行差一格沒消，眼角餘光一直瞄著那行；下一塊剛好補上 → 多巴胺炸裂。這是排行第一的鉤子。 |
| **自我歸因（我變強了）** | 沒有骰子、沒有隨機傷害，輸了只怪自己「那步放錯了」。下一局的動機：「這次我不會再犯。」 |
| **指數型連消獎勵** | 一次消兩行=分數 ×2，消三行+兩列=分數暴衝。玩家主動追求「大消」，自然延長局時。 |
| **損失趨避（格子快滿）** | 看著 8×8 只剩零星空格時，心跳加速。大腦優先「救場」而非享樂，注意力高度集中 = 體感時間縮短 = 根本不想放下。 |
| **收集完整感** | 三個備選方塊像「牌手」，大腦想把三張牌都「用掉」——空槽觸發強迫感。 |

## 難度與節奏（新手活過前 15 秒 / 老手 2 分鐘不無聊）

**新手前 15 秒**：8×8 空格超多，隨便放都能消，初期必得幾次「小勝」，建立「這遊戲我玩得來」的錯覺，留住人。

**中期 30–90 秒**：備選方塊開始出現「難用的L形/S形」，格子填滿速度超過消除速度，玩家感受到真實壓力但還有救。

**老手 2 分鐘後的樂趣**：學會「留出連消路線」，主動設局，把 near-miss 變成可控的連鎖——技術天花板夠高，玩家永遠感覺「再練就能更好」。

---

## 可偷的設計（我們做得到的 3–5 條）

1. **三選一備料區（無時限）**：玩家不被催，焦慮來自格子而非計時器。我們可以把「三個方塊」換成「三隻史萊姆貓形狀的積木」——放進格子就會擺動發出可愛音效。

2. **行列同時消除**：放滿行 + 放滿列同時爆掉，比純俄羅斯方塊更立體的「啊哈瞬間」。純前端邏輯，極易實作。

3. **Near-miss 視覺強調**：快滿的格子換顏色（例：史萊姆貓快被壓扁 → 臉開始變紅）。製造危機感不需要任何後端。

4. **一鍵重試 + 最佳分數**：Game over 畫面只有「再來一局」和「本次分：XXX / 最高分：XXX」，不問帳號不要廣告。摩擦越少，重試率越高。

5. **連消音效疊加**：消一行音效 A，同時消兩行音效 B（A + 高八度），同時消三條音效 C（和弦 + 史萊姆貓大叫）。聽覺獎勵比視覺獎勵更本能。

## 不可行的部分（原作有但我們該捨棄的）

| 原作功能 | 為什麼捨棄 |
|---|---|
| 每日任務 / 挑戰模式（需伺服器） | 我們是靜態網頁小遊戲，沒後端，做了是假的。 |
| 全球排行榜 | 同上；若真要排行，用 localStorage 只存本機紀錄即可，無需伺服器。 |
| 道具系統（炸彈、魔法錘） | 引入道具需要設計平衡、避免玩家感覺被收割，複雜度超出單檔遊戲範圍。 |
| 豐富的視覺主題包 | 美術資源量大。我們一套史萊姆貓主視覺，保持統一宇宙風格即可。 |

## 我們的變形版一句話企劃

> **把史萊姆貓形狀的積木塞進貓窩格子裡，拼滿一行整排喵的爆開 —— 格子越來越滿、貓臉越來越臭，「差一格就消了！」的煎熬感就是整個遊戲的靈魂。**

核心保留：**near-miss 驅動的自我歸因爽感**（不是運氣好，是我放對了）。史萊姆貓宇宙貢獻：每次消除時貓從格子裡彈飛，連消時貓疊羅漢叫成一團。
