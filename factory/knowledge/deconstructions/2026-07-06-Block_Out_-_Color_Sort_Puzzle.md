# 解構：Block Out! - Color Sort Puzzle

## 核心迴圈（一圈幾秒？操作→回饋→獎勵怎麼轉？）

一圈約 10–20 秒。操作只有一種：手指滑動色塊，塊沿格線衝到底（像冰壺），撞牆為止。回饋立即：同色塊相撞就消除，動畫音效炸開。獎勵是「盤面空出來」——每次消除讓可能性突然打開，就是多巴胺峰值。迴圈節奏：看盤→找下一步→滑→咚！消除→盤面重排→再看盤。無等待、零技能門檻、幾乎全程手感驅動。

## 上癮機制

1. **歸因於己（內部控制感）**：每次失敗都是「我沒看清楚」，不是隨機掉落；玩家確信下一局更有把握，反射性按「再玩一次」。
2. **Near-miss**：差一步就能清盤的局面反覆出現，大腦解讀為「幾乎贏了」，比乾脆輸更刺激、更難放下。
3. **視覺整潔渴望（Zeigarnik Effect）**：雜亂盤面在心理上是「未完成任務」，強迫症般想清乾淨，這份不適感驅動連局。
4. **漸進難度的假安全感**：前幾關 3 步秒過，玩家建立「我很厲害」的自我形象；難度陡升後捨不得放棄——沉沒成本加自我形象維護雙重鎖定。
5. **碎片時間完美契合**：一局 30 秒，放下無罪惡感，但又短到「再一局」幾乎是反射動作，完美對齊電梯/等紅燈場景。

## 難度與節奏

**新手存活**：第一關只有兩色四格，一個動作就過；成功動畫強化「我懂了！」的確認感，建立學習信心。

**中段鉤子**：增加顏色與干擾格，但永遠保留至少一個「明顯正確的第一步」，讓玩家不是全面卡死——只是局部糾結，繼續有施力點。

**老手挑戰**：加入固定障礙塊、雙向封死的顏色組合，強迫預判兩步以上，把「反射」升級為「規劃」，挑戰層次徹底不同。

## 可偷的設計（3–5 條，我們做得到的）

1. **「滑到底」慣性感**：不是移動一格，而是衝到撞牆才停——一個操作帶重量感與篤定感，遠比點格子過癮。
2. **消除即獎勵，不需分數**：盤面縮小本身就是正回饋，不用設計複雜積分動畫，視覺爽度自帶。
3. **顏色數量 = 難度旋鈕**：從 2 色到 6 色就是從簡單到地獄，程式成本零，體驗幅度極大，非常適合無限關卡模式。
4. **Near-miss 局刻意設計**：放入「差一步就清盤」的局面作為關卡壓軸，保證再試一次的衝動。
5. **最優解步數顯示**：每關標明「最少 N 步通關」，觸發完美主義——玩家多一個理由重玩，追求效率勝利。

## 不可行的部分

- **百關關卡庫**：原作靠海量手工關卡撐長期留存，網頁單檔沒有維護資源；應改成隨機生成盤面（procedural generation）或計分無限模式。
- **廣告節奏設計**：失敗後插廣告、復活倒計時等變現結構，網頁版完全不適用，拆掉才能保持節奏純粹。
- **特殊道具/鎖定塊系統**：多種特殊磚需要額外教學，30 秒上手的前提下一律砍掉，只留純色塊滑動核心。

## 我們的變形版一句話企劃

把六色史萊姆貓塞進滑滑道，手指一劃讓牠衝到底、碰上同色就黏成一糰噴漿爆開——用最少滑動次數把整個盤面清光，保留「衝撞慣性帶來的篤定手感」與「一次清空的視覺整潔快感」，史萊姆貓的果凍質地讓「相撞黏合」的動畫天然成立。
